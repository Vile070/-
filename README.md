# 権限別ログインシステム

## 概要

このプロジェクトは、**Node.js**、**Express**、**SQLite**、**EJS** を使用して作成した、シンプルな権限別ログインシステムです。

管理者ユーザーと一般ユーザーの2種類のユーザーを用意し、ログイン後にユーザーの権限に応じて異なる画面へ遷移する仕組みを実装しています。

このプロジェクトでは、Webアプリケーションにおけるログイン機能、セッション管理、データベース接続、パスワードのハッシュ化、権限管理の基本的な仕組みを学習することを目的としています。

---

## 主な機能

* ユーザーログイン機能
* bcrypt によるパスワードのハッシュ化
* セッションによるログイン状態の管理
* ユーザー権限による画面遷移
* 管理者画面
* 一般ユーザー画面
* 管理者専用ページへのアクセス制御
* SQLite データベースの利用
* シンプルな画面デザイン

## 動作確認方法

### 管理者ユーザーの場合

1. ログイン画面を開きます。
2. 以下の管理者アカウントでログインします。

```text
ユーザー名：admin
パスワード：admin123
```

3. ログイン後、管理者画面に遷移します。
4. 管理者画面では、ユーザー一覧を確認できます。

---

### 一般ユーザーの場合

1. 管理者アカウントからログアウトします。
2. 以下の一般ユーザーアカウントでログインします。

```text
ユーザー名：user01
パスワード：user123
```

3. ログイン後、一般ユーザー画面に遷移します。
4. 一般ユーザーは管理者画面にアクセスできません。

---

### アクセス制御の確認

一般ユーザーでログインした状態で、以下のURLにアクセスします。

```text
/admin
```

一般ユーザーには管理者権限がないため、アクセスが拒否されます。

## 実装内容

### ログイン認証

SQLite データベースに保存されたユーザー情報をもとに、ユーザー名とパスワードを確認します。

パスワードは bcrypt を使用してハッシュ化しています。

---

### セッション管理

ログイン成功後、ユーザー情報をセッションに保存します。

これにより、ログイン状態を保持し、ログインしていないユーザーが直接ページにアクセスできないようにしています。

---

### 権限管理

ユーザーの権限に応じて、ログイン後の遷移先を分けています。

* 管理者ユーザー：管理者画面へ遷移
* 一般ユーザー：一般ユーザー画面へ遷移

また、一般ユーザーが管理者画面にアクセスしようとした場合は、アクセスを拒否するようにしています。
